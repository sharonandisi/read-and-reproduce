
# read-and-reproduce/README.md

A repository dedicated to reading, understanding, and reproducing foundational AI research papers.

## Purpose

* Build a habit of deeply understanding AI research.
* Recreate simplified versions of important models.
* Document insights, experiments, and challenges.

## Structure

* `papers/` — Notes and breakdowns of each paper.
* `implementations/` — Reproduced code (tiny versions).
* `experiments/` — Tests, visuals, and results.

## Weekly Workflow

* Select 1 paper.
* Read & summarize.
* Reproduce core idea.
* Write reflection.
